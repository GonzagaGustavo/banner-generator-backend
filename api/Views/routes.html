Documentação do projeto

"/"

Essa é a rota principal, se você já está logado lhe redireciona para a rota "/painel", se não estiver logado ficara nesta rota para fazer o login.

"/painel"

Essa é a rota para você criar seu banner.

"/admin"

Essa é a rota do super Admin e do admin que da para ver uma lista de todos os usuários.

"/signup"

Essa é a rota para criar um usuario.

"/admin/edit/:id"

Essa é a rota para editar um usuario. No lugar de (:id) você coloca o id de um usuario, mas somente um super admin ou um admin com permissão conseguira editar.
